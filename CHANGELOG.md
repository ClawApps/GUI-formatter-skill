# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/),
and this project adheres to [Semantic Versioning](https://semver.org/).

## [0.8.0] - 2026-02-08

### Added
- Initial public release
- 12 UI component types: Markdown, Collapse, AppCard, Form, Card, Modal, Alert, Progress, VideoPlayer, AudioPlayer, ImageGallery, WebView
- Intent-based component mapping (13 intent types)
- 3-round validation with automatic Markdown fallback
- 12 form field types with validation rules
- 6 action types (api_call, navigate, emit_event, open_modal, close_modal, reset)
- CLI support (stdin/file input, strict mode, fallback toggle)
- Python library API (format_reply, format_form, format_code, format_alert, etc.)
- Bilingual documentation (English + Chinese)
- Component catalog reference with detailed props and examples
