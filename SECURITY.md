# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in GUI Formatter, please report it responsibly.

**Do NOT open a public issue for security vulnerabilities.**

Instead, please email us at: **dev@clawapps.com**

Include the following in your report:

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

## Response Timeline

- **Acknowledgment**: Within 48 hours
- **Initial assessment**: Within 1 week
- **Fix release**: Depending on severity, typically within 2 weeks

## Scope

This policy applies to the GUI Formatter Skill codebase, including:

- `scripts/` - Python formatting engine
- `SKILL.md` - Skill definition
- `references/` - Documentation

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.8.x   | Yes       |
| < 0.8   | No        |
